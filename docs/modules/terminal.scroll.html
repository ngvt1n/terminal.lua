Module terminal.scroll

Module to manage scroll-regions and scrolling.

Provides functions to manage scroll-regions and perform scrolling operations in the terminal.

Functions

scroll ([n=0]) Scrolls vertically (up/down) and writes it to the terminal.
scroll_apply () Applies the scroll region at the top of the stack and writes it to the terminal.
scroll_applys () Returns the ANSI sequence of the current scroll region (without writing).
scroll_down ([n=1]) Scrolls the screen down and writes it to the terminal.
scroll_downs ([n=1]) Creates an ANSI sequence to scroll down without writing to the terminal.
scroll_pop ([n=1]) Pops n scroll regions off the stack and writes the result to the terminal.
scroll_pops ([n=1]) Pops n scroll regions off the stack (without writing).
scroll_push ([top[, bottom]]) Pushes a new scroll region onto the stack and writes it to the terminal.
scroll_pushs ([top[, bottom]]) Pushes a new scroll region onto the stack without writing it.

scroll_regions ([top[, bottom]]) Creates an ANSI sequence to set the scroll region without writing.
scroll_up ([n=1]) Scrolls the screen up and writes it to the terminal.
scroll_ups ([n=1]) Creates an ANSI sequence to scroll up without writing to the terminal.
scrolls ([n=0]) Creates an ANSI sequence to scroll vertically without writing to the terminal.
